Basics

Java is a class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is a general-purpose programming language intended to let application developers.

Whatever the java program we write that is known as source code and this source code is not understand by our system.

To make our code understandable to our system the java provides two software:

  • compiler
  • jvm
    • (java vrtual machine)

Compiler check the syntax of the program and generate bytecode.

Bytecode is not the machine level language and it is responsible for java to become platform independent.

Jvm converts the bytecode to the machine level language.

    Methods in java are used for to write business logic and application logic.
    Methods name must start withlower case when method name contains more than one word then every inner word should start with uppercase.
    Example : 
                main()                                  toLowerCase();
            append()                                toUpperCase();

    every method contains two parts :
    1.declaration
    2.implementation

    Their are two types of Methods:
    1.static
    2.non-static

                public static void m1(){
                      //  this is a static method , the method which has a static keyword inside the method is called static method.
            }

            public void m1(){
                      //  the method without static keywordis called non-static method.
            }

    Method calling

    calling a method inside the main method is called method calling.
    We can write any number of methods inside the class. 
                class first {
	            public static void main(String[] args) {
		            System.out.println("hi");
		            m1();				//method calling m1()inside the main method
		            System.out.println("bye);	}
	            }
                public static void m1(){
	                System.out.println("hi");  //secondary method
	                System.out.println("bye");
	            }
            }
    We can call static method inside a static area directly.
    We can't call a non-static method inside a static area directly so for that we have to create a object reference. 
                first f=new first();  // with the help of this we can access the non-static method inside a static area directly.
    We can call a non-static method inside a non-static area directly.
    Rules for the method calling
    • inside non-static
      • We can call static and non-static directly
    • inside static
      • we can call a static directly but calling non-staic with the help of object reference

    • Flow control

    The order in which all the statement will be executed at the run time is known as flow control.

    (if....else....) sytnax 
                if(condton){
                statement 1;
            }else{
                statement 2;
            }

    if the conditon s true statement 1 will be executed otherwse the statement will executed
    The condition can be a boolean type otherwise it will shows a compile time error,saying incompatible error, and the statement must not be of declaretive type

    Examples: 
                int x=10;
            if(x){
                System.out.println("hi");               this will gives you an error
            }else{
                System.out.println("bye");
            }

            int x=10;
            if(x=20){
                System.out.println("hi";);              this will also gives you an error
            }else{
                System.out.println("bye");
            }

            int x=10;

                System.out.println("hi";);              this will gves you an output

                System.out.println("bye");
            }

    Else part is optional, the code should run otherwse it will goes out of the condition and print nothing.

    Switch Case

    syntax 
                switch(condition){
                case 1:
                statement1;
                break;
                case 2:
                statement2;
                break;
                case 3:
                statement3;
                break;
            }

    The syntax is compile but not shows an output because both case and default are optional in java.An anti-switch case s a valid java program.

            }

                int x=20;
            switch (x){

    Every where in java Curly crackets are optional but not in switch case, we must have a curly brackets to perform switch case,Case value must be a constant expression not a variable.

                int x=10;

            switch(x){

                break;
                case y:                             Error: constant is required

                int x=97;

    Fall back inside switch

    Whenever a case matches from that case unwards all the statement will be exected untill break or end of the switch. this is known as fall back inside switch

                switch(x){
                case 0:
                System.out.println(0);
                case 1:
                System.out.println(1);
                break;
                case 2:
                System.out.println(2);
                default:
                System.out.println("default");

            }

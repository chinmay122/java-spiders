    Bitwise Operator

    • a & b -- it returns true if both arguments are true.
    • a | b -- it returns true if any one of the argument is true.
    • a ^ b -- it returns true if both arguments are different.

    Exapmples:

     
            System.out.println(true & false); (false)
            System.out.println(true | flase);  (true)
            System.out.println(true ^ false);  (true)

    Logical operator (Short circuit operator)

                a && b  -- b will be executed when the a is true.
            a || b  -- b will we executed when the a s false.

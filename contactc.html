    for...loop...

    when we know the exact number of iteration then we should go for the "for...loop..."

    Syntax 
                for(initializaton;condition;increment/decrement){
                statement;
            }

    1.Intialization

    The initialization part will be executed onlu once in the loop.

    We can declare any number of variable in initialization part but we have to declare the variable of some type

    Examples 
                int i=0,j=0,k=0;
        
                int i=0;                                            this ell gives you an output.there are no mandatory that you have to initialize the variable at onlu inside the for loop.
            for(System.out.println("hi") ; i < 10 ; i++){
                System.out.println("bye";)
            }

    We can write statement in the intializaton part.

    2.Condition part

    The condition part is not mandatory in for loop, but bydefault it shows the conditon always true. 
                for(int i=0; ;i++){
                System.out.println("hi");
            }           int output it will run the code infinite times.

    3.Increment and Decrement part

                for (int i=0;i< 10; System.out.println("bye")){
                System.out.println("hi");
                i++;
            }
    All the three parts are optional in for loop and independent of each other.

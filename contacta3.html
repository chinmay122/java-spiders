Type casting

Converting from one type to another type is called type casting.

    Their are two types of type castng :
  • Implicite type casting

    • Complier is responsible to perform type casting
    When ever we are assigning a smaller value to a bigger datatype variable implicite type casting automatically perform
    It is also known as upcasting and widening

                    double d=10;
                System.out.println(d);
  • Explicite type casting

    • Programmer is responsible to perform explicite typr casting.
    When ver we are assgning a bigger data value to a smaller datatype variable explicite type costng will be occur.
    it is also known as down casting or narrowing. Their may be some loss of information.

            int x=10;
